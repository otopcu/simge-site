# SimGe

**SimGe (Simulation Generator)** is a High Level Architecture (HLA) object model editor, federation modeling environment, and code-generation tool. It fully supports the evolving IEEE 1516 standards, including IEEE 1516-2025 (HLA 4), while maintaining backward compatibility with earlier HLA specifications.
